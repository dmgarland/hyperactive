ColorFX class

version: Documentation not provided.

description:

Offers some color effect methods. Used by ColorNegative, ColorBrightness and ColorDodge classes.

Example 1: Sets the movieclip mc to a negative color value. 

			var myColorFX:ColorFX = new ColorFX(mc);
			myColorFX.setNegative(255);

usage:

var myColorFX:ColorFX = new ColorFX(mc);

parameters:

  • mc (MovieClip) Movieclip to animate.
