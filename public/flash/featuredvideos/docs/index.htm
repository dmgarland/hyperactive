AnimationPackage 2 is an ActionScript 2.0 library compatible to Flash Player 8 and above. It helps you to create powerful, maintainable animations and primitive shapes in an efficient and easy way, resulting in small file sizes.

GETTING STARTED: Make sure you check out the support page.

Extra: SuperShapePanel 0.3 is a Flash MX 2004 command that draws a SuperShape right into the Flash IDE. SuperShapePanel.mxp
