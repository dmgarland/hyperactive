Class VideoConversionWorker
In: lib/workers/video_conversion_worker.rb
Parent: BackgrounDRb::Rails

A worker class to convert uploaded videos. Currently it converts any uploaded video into an FLV, an OGG, and leaves the original in place. It also generates a .torrent file which points to the OGG, and adds the torrent to the list of active torrents held by the TorrentWorker.

It is necessary to have ffmpeg and ffmpeg2theora installed to get conversion to work. Torrent creation requires that the bittornado package is installed.
