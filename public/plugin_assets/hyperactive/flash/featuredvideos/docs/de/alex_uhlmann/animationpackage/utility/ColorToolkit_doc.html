ColorToolkit class

version: Documentation not provided.

description:

Offers easy access to properties of the Color transform object. Used by ColorTransform and ColorDodge classes.

Example 1: Sets the movieclip mc to a solid write color. 

			var myColorToolkit:ColorToolkit = new ColorToolkit(mc);
			myColorToolkit.setColorOffsetHex(0xffffff);

usage:

var myColorToolkit:ColorToolkit = new ColorToolkit(mc);

parameters:

  • mc (MovieClip) Movieclip to animate.
